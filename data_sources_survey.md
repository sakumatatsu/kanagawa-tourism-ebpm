# 神奈川県観光振興分析 民間データソース調査レポート

**調査日**: 2026年2月22日
**調査範囲**: 民間企業・準公的機関が公表する観光分析活用可能データ
**対象地域**: 神奈川県
**注記**: 本レポートは2025年5月時点の知識に基づく。各URLの有効性・最新の料金体系は変動の可能性があるため、利用時に公式サイトで確認すること。

---

## 1. 人流データ・位置情報データ

### 1-1. V-RESAS（V-リーサス）

| 項目 | 内容 |
|------|------|
| **サービス名** | V-RESAS（V-リーサス） |
| **提供企業** | 内閣府（データ提供元: Agoop、ナウキャスト等の民間企業） |
| **URL** | https://v-resas.go.jp/ |
| **サービス状況** | **2023年3月末にサービス終了**。現在はアーカイブデータのみ閲覧可能な場合あり |
| **データ内容** | COVID-19影響下の地域経済動向を可視化。人流の増減（対2020年1月比）、飲食店情報の閲覧数、宿泊者数の推移、消費動向等 |
| **神奈川県データ** | 都道府県単位・市区町村単位で取得可能だった |
| **利用条件** | 無料（公開終了済み） |
| **データ粒度** | 都道府県・市区町村単位、週次更新だった |
| **観光分析ユースケース** | COVID-19からの回復期における人流変化の時系列分析、地域間比較。ただし既にサービス終了のため、過去データの参照用途に限定される |
| **備考** | 後継としてRESASの機能拡充が行われている |

### 1-2. RESAS（リーサス：地域経済分析システム）

| 項目 | 内容 |
|------|------|
| **サービス名** | RESAS（地域経済分析システム） |
| **提供企業** | 内閣府 まち・ひと・しごと創生本部（データ提供: 帝国データバンク、NTTドコモ等） |
| **URL** | https://resas.go.jp/ |
| **データ内容** | **観光マップ**として以下を提供: (1) 目的地分析（From-to分析）: 観光客がどこから来ているか、(2) 外国人訪問分析: 国籍別の訪問先、(3) 宿泊施設の稼働分析、(4) メッシュ別の人口流動 |
| **神奈川県データ** | 全市区町村のデータが取得可能 |
| **利用条件** | 無料。RESAS-API（要登録）でデータ取得可能 |
| **API** | あり（https://opendata.resas-portal.go.jp/ ）。APIキー取得後、JSON形式でデータ取得可能 |
| **データ粒度** | 市区町村単位。観光マップは年次・月次データあり |
| **更新頻度** | 年1-2回（元データの公表に依存） |
| **観光分析ユースケース** | 市町村別の観光客発地分析、インバウンド訪問先分析、滞在時間分析。**ダッシュボードの「切り口D:旅行者属性別」に直結** |
| **RESAS観光マップの主要機能** | |
| - 目的地分析 | 特定市区町村への訪問者の居住地分布を可視化 |
| - 周遊分析 | 観光客がどの順番でどのスポットを周遊しているか |
| - 滞在人口 | メッシュ単位の滞在人口の時間帯別変動 |
| - 外国人メッシュ | 外国人の訪問エリア分布 |

### 1-3. Agoop（アグープ）流動人口データ

| 項目 | 内容 |
|------|------|
| **サービス名** | 流動人口データ / ポイント型流動人口データ |
| **提供企業** | 株式会社Agoop（ソフトバンクグループ） |
| **URL** | https://www.agoop.co.jp/ |
| **データ内容** | スマートフォンの位置情報データを統計処理した人流データ。特定エリアへの来訪者数、滞在時間、発地分析（どこから来たか）、時間帯別の人口分布 |

| **利用条件** | **有料**（法人向け）。個別見積もり。自治体向けプランあり |
| **API** | あり（法人契約者向け） |
| **データ粒度** | 125mメッシュ / 250mメッシュ / 500mメッシュ、時間帯別（1時間単位） |
| **更新頻度** | 日次〜月次（契約プランによる） |
| **観光分析ユースケース** | 観光スポットごとの来訪者数推計、イベント開催時の人流変化測定、周遊ルート分析、時間帯別の混雑状況把握。**ダッシュボードの地域別分析・イベント効果測定に最適** |
| **備考** | V-RESASにもデータ提供実績あり。自治体の観光計画策定での利用実績多数 |

## 10. 注意事項・留意点

### データ利用に関する留意点

1. **個人情報保護**: 人流データ・決済データは統計処理済みのデータを使用すること。個人を特定できるデータの取り扱いは個人情報保護法の規制対象
2. **利用規約の遵守**: 各データソースの利用規約を確認し、目的外利用・再配布の制限に注意
3. **データの時間軸の統一**: 年次データ・月次データ・リアルタイムデータが混在するため、分析時の時間軸の統一に留意
4. **推計値の限界**: 人流データ・決済データはサンプルデータからの推計値であり、実数ではない点を明記
5. **COVID-19の影響**: 2020-2022年のデータはCOVID-19の影響で異常値が含まれる可能性。基準年の設定に注意
6. **V-RESASのサービス終了**: 2023年3月末にサービスが終了しているため、過去データの参照のみ可能。代替としてRESASの機能を活用すること

### データ更新・サービス変更リスク

- APIの仕様変更・廃止が発生する可能性がある
- 有料サービスの料金改定が行われる可能性がある
- URLは変更される可能性があるため、定期的な確認を推奨
- 本レポートは2025年5月時点の情報に基づいているため、各サービスの最新状況は公式サイトで確認すること
