## 9. RESAS（resas）

### 概要
- **提供**: 内閣府
- **URL**: https://resas.go.jp/
- **API**: https://opendata.resas-portal.go.jp/

### 取得状況
**メタデータのみ取得**。実データ取得にはRESAS-APIキー（無料登録）が必要。

### 取得可能なデータ
- 目的地分析（From-to分析）: 神奈川県各市町村への来訪者居住地分布
- 周遊分析: 観光スポット間の周遊パターン
- 滞在人口分析: メッシュ単位の時間帯別滞在人口
- 外国人訪問分析: 国籍別の訪問先分布

## データ取得における制約と今後の対応

### 制約事項
1. **ネットワークアクセス不可**: 本収集はトレーニングデータに基づくため、2025年5月以降に更新されたデータは反映されていない。
2. **概数ベース**: 家計調査・消費者物価指数の値は概数であり、正確な値はe-Statからの直接取得が必要。
3. **e-Stat API未接続**: APIキーを用いた自動データ取得は未実施。
4. **RESAS未接続**: APIキー取得・データ取得は未実施。
5. **MICE統計の欠損**: 2023年以降のJNTO MICE統計は公表タイミングの関係で未取得。

### 今後の対応推奨事項

| 優先度 | 対応事項 | 方法 |
|--------|---------|------|
| **最高** | e-Stat APIで宿泊旅行統計の最新月次データを取得 | e-Stat API（appId取得→getStatsData呼出、cdArea=14） |
| **最高** | JNTO公式サイトで2024年確定値・2025年速報値を確認 | https://www.jnto.go.jp/statistics/ |
| **高** | JNTO MICE統計の最新公表データを確認 | https://www.jnto.go.jp/statistics/data/conference-statistics/ |
| **高** | 神奈川県入込観光客調査の最新年度版を取得 | 神奈川県文化スポーツ観光局ページ |
| **高** | RESAS-APIキーを取得し、From-to分析データを取得 | https://opendata.resas-portal.go.jp/ |
| **中** | e-Stat APIで家計調査・CPI の正確な値を取得 | e-Stat API（品目コード指定） |
| **中** | 観光庁の訪日外国人消費動向調査の神奈川県訪問率最新値を確認 | e-Stat API |
| **低** | 旅行・観光サテライト勘定の最新版確認 | 観光庁ページ |
