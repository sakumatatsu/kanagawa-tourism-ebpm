# 神奈川県観光振興EBPMダッシュボード 構築計画

## 1. プロジェクト概要

### 目的
新かながわグランドデザインの観光振興施策を因数分解し、各構成要素に対応するデータを可視化することで、EBPM（証拠に基づく政策立案）を支援するインタラクティブダッシュボードを構築する。

### コンセプト
**「観光振興という最上位目標から、思考フローに沿って段階的にドリルダウンしていく」**

ユーザーが分解の切り口を選択しながら、最上位のアウトカム→中間アウトカム→アウトプット→個別データへと掘り下げていくUI。

### 技術方針
- Phase 1: HTML/CSS/JS スタンドアロン（ブラウザで開くだけで動作）
- 将来: BIツール連携、クラウドホスティング対応

## 2. 因数分解の構造設計

### 2-1. ロジックモデル（アウトカム→施策→データの階層）

```
Level 0: 最上位アウトカム
  「観光振興による地域経済の持続的発展」
  └ 指標: 観光消費額総額（目標: 1兆1,896億円）

Level 1: 分解の切り口（ユーザーが選択）
  ├── [A] 施策・事業別（新かながわグランドデザイン準拠）
  ├── [B] 地域別（5地域政策圏 + 詳細7エリア）
  ├── [C] 観光資源タイプ別
  └── [D] 旅行者属性別

Level 2: 各切り口の中間アウトカム
  （以下詳細）

Level 3: アウトプット指標・個別データ
  （各中間アウトカムに紐づくKPI・統計データ）
```

### 2-2. 切り口A: 施策・事業別（最優先）

観光振興（PJ5 観光・地域活性化）

│   │   └── データ: 外国人対応施設数、Wi-Fi整備率、キャッシュレス対応率
│   └── 施策320: 戦略的プロモーション
│       ├── KPI: 海外向けSNSフォロワー数（218,789 → 295,000）
│       └── データ: プロモーション実施件数、メディア露出数
├── 柱B: 地域にひとの流れをつくる取組

│   │   └── KPI: 三浦半島消費額（183億円 → 265億円）

│   │   └── データ: 海洋ツーリズム参加者数
│   ├── 県西地域活性化PJ
│   │   ├── KPI: 県西地域入込観光客数（3,315万人 → 3,655万人）

### 4-2. インタラクション設計

1. **初期表示**: 最上位アウトカム（観光消費額・入込客数）のサマリー
2. **切り口選択**: 4つのタブから分解軸を選ぶ
3. **ドリルダウン**: ツリーマップの要素をクリックすると下位階層へ
4. **パンくずリスト**: 現在の階層位置を表示、上位に戻れる
5. **詳細パネル**: 選択した要素のKPI・データをグラフで表示

### 4-3. ビジュアライゼーション手法

| 用途 | チャートタイプ | ライブラリ |
|------|-------------|-----------|
| 全体構造の俯瞰 | サンバースト（太陽光）チャート | D3.js |
| 構成比の可視化 | ツリーマップ | D3.js |
| KPI達成状況 | ゲージチャート / ブレットチャート | Chart.js |
| 時系列推移 | 折れ線グラフ | Chart.js |
| 地域比較 | 棒グラフ / レーダーチャート | Chart.js |
| 地理的分布 | 神奈川県地図（コロプレス） | D3.js + GeoJSON |

## 5. 技術スタック

| レイヤー | 技術 | 理由 |
|---------|------|------|
| フレームワーク | なし（Vanilla JS） | スタンドアロン要件、依存性最小化 |
| グラフ描画 | D3.js v7（CDN） | サンバースト/ツリーマップに最適 |
| 補助グラフ | Chart.js v4（CDN） | 棒・折れ線・ゲージに簡便 |
| スタイリング | CSS3（カスタムプロパティ） | レスポンシブ対応 |
| データ形式 | JSON（埋め込み） | 外部サーバー不要 |
