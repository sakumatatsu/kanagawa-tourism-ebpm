## 調査概要

- **調査日**: 2026年2月22日
- **調査対象**: e-Stat（政府統計の総合窓口）で取得可能な、神奈川県の観光振興分析に活用できる統計データ

## 1. 観光・宿泊に直接関連する統計

### 1-1. 宿泊旅行統計調査

| 項目 | 内容 |
|------|------|
| **統計名** | 宿泊旅行統計調査 |
| **調査主体** | 観光庁（国土交通省） |
| **e-Stat統計ID** | 00604010 |
| **神奈川県データ** | **取得可能** |
| **データ粒度** | 都道府県レベル（一部は観光地タイプ別も可能） |
| **更新頻度** | 月次（速報）、年次（確報） |
| **データ形式** | CSV、Excel（e-Stat DB形式） |
| **e-Stat API** | **利用可能**（statsDataId指定で取得可） |
| **主な収録項目** | 延べ宿泊者数、実宿泊者数、外国人宿泊者数、客室稼働率、施設タイプ別宿泊者数、国籍別外国人宿泊者数 |
| **観光分析での活用** | 宿泊需要の把握、インバウンド動向分析、季節変動分析、施設タイプ別需要分析 |

**補足**:
- 従業員数10人以上の施設は全数調査、10人未満はサンプル調査
- 2007年（平成19年）からデータ利用可能
- 月次データにより季節変動・イベント効果の分析が可能
- 国籍別データにより、インバウンド戦略のターゲット分析が可能

## 10. 総括：推奨データ取得優先順位

### 最優先（観光振興の核となるデータ）
1. **宿泊旅行統計調査** - 最も直接的な観光指標、月次で動向把握可能
2. **訪日外国人消費動向調査** - インバウンド戦略の基礎データ
3. **経済センサス** - 観光産業の構造を市町村レベルで把握
4. **旅客地域流動調査** - 来訪者の流動パターン把握

### 高優先（分析の深化に必要）
5. **旅行・観光消費動向調査** - 消費額推計の基礎
6. **県民経済計算** - 経済規模の把握
7. **国勢調査** - 小地域の人口・産業構造
8. **社会教育調査** - 文化施設の集積・利用

### 中優先（補助的データ）
9. **家計調査** - 消費者側の旅行支出
10. **鉄道輸送統計** - 交通利用の詳細
11. **港湾統計** - クルーズ観光
12. **社会生活基本調査** - 余暇行動の実態
13. **就業構造基本調査** - 雇用の質的分析

### 参考（特定分析時に活用）
14. 消費者物価指数、毎月勤労統計、建築着工統計、出入国管理統計、宗教統計調査 等
